20 hrs

Introduction to Containers

Subtopics:

  • Basics of Containerization (4 hours): Understand what containers are and why they are used.
  • Docker Essentials (8 hours): Learn how to install Docker, run Docker containers, create Docker images, and manage Docker volumes and networks.
  • Docker Compose (4 hours): Introduction to orchestrating multi-container applications.
  • Hands-On Projects (4 hours): Practice by containerizing a simple web application.

20 hrs

Introduction to Container Orchestration with Kubernetes

Subtopics:

  • Kubernetes Basics (6 hours): Learn about Kubernetes clusters, pods, deployments, services, and how they interact.
  • Deploying Applications (6 hours): Practice deploying applications on Kubernetes.
  • Scaling and Managing Resources (4 hours): Understand how to scale applications and manage resources in Kubernetes.
  • Hands-On Mini-Project (4 hours): Deploy a multi-container application using Kubernetes.

20 hrs

Introduction to Infrastructure as Code (IaC)

Subtopics:

  • IaC Concepts (5 hours): Learn the principles of Infrastructure as Code.
  • Terraform Basics (7 hours): Introduction to Terraform, including setup, basic commands, and writing Terraform scripts.
  • Managing Cloud Resources (4 hours): Use Terraform to provision and manage resources on a cloud platform (AWS, Azure, or GCP).
  • Hands-On Project (4 hours): Write a simple Terraform script to deploy a basic web application infrastructure.

20 hrs

Continuous Integration and Continuous Deployment (CI/CD)

Subtopics:

  • CI/CD Concepts (4 hours): Understand the basics of Continuous Integration and Continuous Deployment.
  • GitHub Actions (8 hours): Learn how to create workflows that automate your build, test, and deployment processes with GitHub Actions.
  • Integration and Deployment Strategies (4 hours): Learn about different strategies for integration and deployment.
  • Hands-On Project (4 hours): Set up a CI/CD pipeline for a simple web application using GitHub Actions.

20 hrs

Basic Monitoring and Logging

Subtopics:

  • Monitoring Basics (5 hours): Learn the fundamentals of monitoring applications and infrastructure.
  • Introduction to Prometheus and Grafana (7 hours): Get started with Prometheus for monitoring and Grafana for visualization.
  • Logging Basics (4 hours): Introduction to log management concepts and tools (e.g., ELK stack or Loki).
  • Hands-On Project (4 hours): Set up basic monitoring and logging for a web application.

30hrs

Cloud Computing and DevOps Practices in the Cloud

  • Cloud Computing Fundamentals (3 hours): Understand the basics of cloud computing, including its definition, advantages, and service models (IaaS, PaaS, SaaS).
  • Major Cloud Providers Overview (3 hours): Get an overview of AWS, Azure, and Google Cloud Platform, focusing on their key services relevant to DevOps.
  • Cloud Networking Basics (2 hours): Introduction to cloud networking concepts, including VPCs, subnets, and internet gateways.
  • Hands-On Practice (2 hours): Set up a basic cloud environment using the free tier of a cloud provider, such as AWS or Google Cloud.
  • Cloud-based CI/CD Tools (3 hours): Explore cloud-based CI/CD tools and services, such as AWS CodeBuild, AWS CodeDeploy, Azure DevOps, and Google Cloud Build.
  • Using Managed Services for DevOps (3 hours): Learn about using managed services (e.g., database, Kubernetes, and serverless functions) to simplify operations and focus on development.
  • Security and Compliance in the Cloud (2 hours): Basics of cloud security best practices, identity and access management (IAM), and compliance standards.
  • Hands-On Project (2 hours): Deploy a simple application using cloud services, incorporating CI/CD practices and security best practices.
